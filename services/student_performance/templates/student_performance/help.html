Общая информация
Что такое ОРИОКС?

ОРИОКС - это информационная система для управления учебным процессом. Она позволяет студентам:

  • Просматривать свои оценки и успеваемость
  • Отслеживать расписание занятий
  • Контролировать посещаемость
  • Получать уведомления о важных событиях
Как начать работу?
  1. Войдите в систему, используя свои учетные данные
  2. На главной странице вы увидите краткую информацию о вашей успеваемости
  3. Используйте навигационное меню для доступа к различным разделам
Оценки
Как просматривать оценки?

Для просмотра оценок:

  1. Перейдите в раздел "Оценки"
  2. Используйте фильтры для выбора предмета и семестра
  3. Оценки отображаются в виде карточек с детальной информацией
Как рассчитывается средний балл?

Средний балл рассчитывается автоматически с учетом:

  • Весомости различных типов работ
  • Количества оценок по каждому предмету
  • Текущего семестра
Расписание
Как просматривать расписание?

Для просмотра расписания:

  1. Перейдите в раздел "Расписание"
  2. Выберите нужную неделю с помощью календаря
  3. Текущее занятие будет выделено специальным образом
Как узнать о заменах?

Информация о заменах:

  • Отображается в расписании с пометкой "Замена"
  • Дублируется в разделе уведомлений
  • Может быть отправлена на email
Посещаемость
Как отслеживать посещаемость?

Для просмотра посещаемости:

  1. Перейдите в раздел "Посещаемость"
  2. Выберите предмет и период
  3. Статистика отображается в виде графиков и таблиц
Как отметить уважительную причину отсутствия?

Для отметки уважительной причины:

  • Обратитесь к преподавателю
  • Предоставьте необходимые документы
  • Дождитесь подтверждения в системе
Профиль
Как изменить данные профиля?

Для изменения данных:

  1. Нажмите на свое имя в правом верхнем углу
  2. Выберите "Редактировать профиль"
  3. Внесите необходимые изменения
Как настроить уведомления?

Для настройки уведомлений:

  • Перейдите в настройки профиля
  • Выберите тип уведомлений
  • Укажите предпочтительный способ получения

Как сообщить об ошибке?

Для сообщения об ошибке:

  1. Сделайте скриншот проблемы
  2. Опишите последовательность действий
  3. Отправьте информацию на support@orioks.ru
